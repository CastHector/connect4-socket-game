# Multiplayer Connect 4 (Socket Programming)

A client-server multiplayer Connect 4 game built using Java socket programming.

## Technologies
Java
Socket Programming
Java Swing (GUI)

## Features
- Client-server architecture
- Multiplayer gameplay
- Server-managed game sessions
- GUI interface for players

## How to Run
1. Start the server
2. Launch two clients
3. Connect to the server to begin a match
